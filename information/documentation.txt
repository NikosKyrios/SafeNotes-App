SafeNotes 🔐
Secure Note-Taking Application with Advanced Security Features

📋 Overview
SafeNotes is a desktop application for storing encrypted notes with multiple layers of security, including password protection, keystroke dynamics analysis, and optional two-factor authentication. All data is encrypted locally before storage.

✨ Key Features
Local Encryption: AES-256-GCM encryption for all notes

Keystroke Biometrics: Behavioral authentication based on typing patterns

Password Strength Analysis: Real-time password validation and scoring

Security Monitoring: Alerts for suspicious login attempts

Cross-Platform: Works on Windows, macOS, and Linux

🔐 Security Architecture
Encryption & Hashing
Algorithm: AES-256-GCM with 128-bit authentication tags

Key Derivation: Argon2id with 64MB memory usage

Salting: 16-byte random salts for each password

IV Generation: 12-byte random initialization vectors for GCM

Why These Choices?
AES/GCM/NoPadding: Provides both confidentiality and authentication in one operation

Argon2: Memory-hard algorithm resistant to GPU/ASIC attacks

256-bit keys: Future-proof against quantum computing advances

GCM mode: Prevents padding oracle attacks and provides authentication

❓ Frequently Asked Questions


Q: What's the purpose of the keystroke timing feature?
A: Behavioral biometrics. By analyzing typing speed and rhythm, the system can:

Detect automated login attempts

Add an extra layer of authentication

Alert on anomalous behavior

Slow down brute-force attacks

Q: Why clear password fields after login attempts?
A:

Security: Prevents password exposure if someone walks by

Privacy: Shared computers won't show previous passwords

UX: Clear indication that attempt was processed

Q: How does the password strength algorithm work?
A: Each requirement adds 20 points:

✓ 10+ characters (20 points)

✓ Uppercase letter (20 points)

✓ Lowercase letter (20 points)

✓ Digit (20 points)

✓ Special character (20 points)

Score interpretation:

0-39: Very Weak

40-59: Weak

60-79: Good

80-100: Strong
